# nCov Health Tracker
 
An application that will allow government agencies to track person(s) suspected of having coronavirus infection.
#
*Architecture Type: Microservices*

*Technologies*:

    - Container Orchestration
        - Kubernetes
            - Ingress
    - Container Technology
        - Docker
    - Backend
        - Framework
            - Spring Boot
            - Spring Cloud
                - Ribbon
                - Hystrix
    - User Interface
        - Framework
            - React
    - Mobile
        - Android
    - Database
        - PostgreSQL
    - Automation
        - Maven
        - Github Workflow
        - Skaffold
        - Shell script
    - Database Migration
        - Flyway
    - Continous Integration / Deployment
        - Circle CI

*Modules*:

    - Account Service
    - Government Service
    - Hospital Service
    - Infection Service
    - Notification Service
    - Person Service
    - Security Service
